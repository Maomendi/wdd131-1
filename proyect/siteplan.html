Website Planning Document

This document outlines the purpose, structure, audience, design decisions, and visual identity for Adriana's Classroom.

The goal is to create a welcoming and professional language learning website.

Site Name

Adriana's Classroom

This name was chosen because it reflects a warm, personal, and welcoming learning environment. It emphasizes connection between teacher and student while maintaining professionalism.

Site Purpose

The purpose of this website is to provide information about language courses offered by Adriana. The site will include class information, teaching philosophy, cultural learning resources, and contact details for enrollment.

Scenario 1

How can I improve my Spanish speaking skills with a native teacher?

Scenario 2

What teaching methods does Adriana use to make language learning engaging?

Color Scheme

#E6B325 (Warm Mustard) – Used for buttons and accent highlights.

#2F3E75 (Deep Slate Blue) – Used for headings, navigation, and footer.

#FFF9E6 (Soft Cream) – Used as a background for the hero section.

Typography

Playfair Display – Used for headings to create an academic and elegant feel.

Poppins – Used for body text for readability and modern appearance.
